# Altitude and Stabilization Control PID

The name is pretty self-explanatory. it's a control system for quadcopters in create aeronautics. It includes two files:

- **altstabpid.lua** *the control itself - you want to run this code*
- **pid.lua** *the pid code*

>  **pid.lua** wasn't made by me, i took it from [CC: Advanced Math](https://modrinth.com/datapack/advanced_math) 

  Anyone is **welcomed** and **encouraged** to improve this control script. ❗
